# Expense Reimbursement System

A full-stack Employee Expense Reimbursement Management System that digitizes the employee reimbursement process by allowing employees to submit expense claims and enabling HODs, Finance, Accounts, and Admin to review, approve, process, and monitor reimbursements.

## Features

- Role-based Authentication (Employee, HOD, Finance, Accounts, Admin)
- Expense Claim Submission
- Receipt Upload
- Multi-Level Approval Workflow
- Expense History Tracking
- Reports & Analytics Dashboard
- Notifications
- Audit Logs
- Department & Category Management
